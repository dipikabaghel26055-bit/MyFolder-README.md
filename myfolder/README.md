# PHP Development Environment Setup

## Objective
Set up a local development environment for PHP and MySQL and configure GitHub version control.

## Tools Used
- XAMPP
- Visual Studio Code
- Git
- GitHub

## Steps Completed
1. Installed XAMPP and started Apache/MySQL.
2. Installed Visual Studio Code.
3. Installed Git.
4. Created a GitHub repository.
5. Added project files:
   - index.php
   - README.md
6. Made the initial commit and pushed to GitHub.

## Output
Development environment successfully configured.
GitHub repository created with initial commit.
